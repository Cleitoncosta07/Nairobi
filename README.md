## 🤖 Sobre o Projeto

O **Nairobi Bot** é um bot privado de WhatsApp desenvolvido para automação inteligente e eficiente. Inspirado na personagem Nairobi da série *La Casa de Papel*, o projeto combina tecnologia, elegância e estratégia em uma única solução.

Este bot foi criado para uso pessoal e não está disponível para distribuição pública.

---

## ✨ Funcionalidades

- 📱 Automação de mensagens no WhatsApp  
- 🤖 Sistema de comandos interativos  
- 🧠 Integração com Inteligência Artificial  
- 🎵 Download de músicas e vídeos  
- 🖼️ Criação e conversão de figurinhas  
- 🔎 Pesquisas automáticas na internet  
- ⏰ Agendamento de tarefas  
- 📊 Respostas rápidas e personalizadas  
- 🔐 Sistema seguro e otimizado  
- 🚀 Estrutura moderna e escalável  

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Descrição |
|------------|-----------|
| ⚡ Node.js | Ambiente de execução JavaScript |
| 📡 Baileys | Integração com o WhatsApp |
| 💻 JavaScript (ES Modules) | Estrutura moderna |
| 🌐 Axios | Requisições HTTP |
| ⏰ Node-Cron | Agendamento de tarefas |
| 🎬 FFmpeg | Processamento de áudio e vídeo |
| 🖼️ Jimp | Manipulação de imagens |
| 📊 Pino | Sistema de logs |
| 📱 QRCode-Terminal | Autenticação via QR Code |
| 🎧 YT-Search | Busca de conteúdos no YouTube |

## 🚀 Instalação

Siga os passos abaixo para instalar e executar o Nairobi Bot.

### 📥 1. Clonar o Repositório
```bash
git clone https://github.com/Cleitoncosta07/Nairobi.git
```

### 📂 2. Acessar a Pasta do Projeto
```bash
cd Nairobi
```

### 📦 3. Instalar as Dependências
```bash
npm install
```

### ▶️ 4. Iniciar o Bot
```bash
npm start
```

Após iniciar, escaneie o QR Code com o WhatsApp para conectar o bot.

---

### 💻 Modo de Desenvolvimento (Opcional)
```bash
npm run dev
```

---

### ⚙️ Configuração Inicial (Opcional)
```bash
npm run config
```

Para instalar automaticamente os recursos necessários:
```bash
npm run config:install
```

---

### 🔄 Atualizar o Bot (Opcional)
```bash
npm run update
```
